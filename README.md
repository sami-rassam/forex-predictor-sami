\# Forex Currency Prediction App



This is an end-to-end machine learning project that forecasts foreign exchange rates using historical currency data.



\## Project Overview



This project includes:



\- Data cleaning and preprocessing

\- Exploratory data analysis

\- Feature engineering for time-series forecasting

\- Machine learning model training

\- Model evaluation using MAE, RMSE and MAPE

\- Saved models for each currency

\- A Streamlit web application for interactive forecasting
